# OpenGL_CW
This project demonstrates a simple 3D animation using OpenGL and GLUT in C++. It renders a rotating 3D teapot with lighting effects on a blue background.  The program uses OpenGL transformation and animation techniques to continuously rotate the teapot around the Y-axis.

This project is useful for learning basic computer graphics concepts such as:

3D object rendering
Lighting
Camera positioning
Animation using timers
Depth testing

Technologies Used

C++
OpenGL
GLUT (OpenGL Utility Toolkit)
